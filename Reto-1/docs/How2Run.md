# How 2 run this thing

## Secuential 
```
gcc -Wall -O0 -lm ./Reto-1/src/JacobiSecuential.c -o secuential
```

-Wall: all warnings.
-O0: No optimizations.
-lm: To use math.h in C.

## Memory 
```
gcc -Wall -O0 -lm ./Reto-1/src/JacobiMemory.c -o memory
```

-Wall: all warnings.
-O0: No optimizations.
-lm: To use math.h in C.
